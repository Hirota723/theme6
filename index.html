実験の目的

Web の仕組みを理解し、Web ドキュメントの文書データ管理の概念と共に、人が見る・使うという観点から文書データのデザインの概念についても理解を深める。 また、Web 文書の記述フォーマットに対するアクセシビリティチェックのツール等を使い、様々な Web サイトやページ対して Web ユーザビリティや Web アクセシビリティの調査を行い、考察する。そして、Web による外部デバイス操作の仕組みを理解する。
